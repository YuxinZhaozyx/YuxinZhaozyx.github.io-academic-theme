GoMicroBookshop

Yuxin Zhao 赵煜新
Jul 4, 2019 project
Code
Abstract Structure

A bookshop project for my go-micro and micro service learning.

本项目依据项目 microservice-in-cn 学习micro工具链以及微服务。

项目简介

本项目为一个网上书店。

业务模块

  • 用户，users
  • 库存，inventory
  • 权限，auth
  • 订单，orders
  • 支付，payment

服务抽象结构

用户、订单、支付服务都会有对外暴露接口，故而它们各自有web层。web层app之间不会互相调用，它们只会与非web层的应用交互。
